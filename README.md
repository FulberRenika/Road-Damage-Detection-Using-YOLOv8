# Road Damage Detection Using YOLOv8

## Overview

This project detects road damages such as cracks and potholes using YOLOv8.

## Dataset

RDD2022 Road Damage Dataset

## Classes

- D00
- D01
- D10
- D11
- D20
- D40
- D43
- D44

## Technologies Used

- Python
- YOLOv8
- Google Colab
- OpenCV

## Training

Model: YOLOv8n

Epochs: 20

Image Size: 640

Batch Size: 16

## Results

The trained model successfully detects various road damages from road images.

- Real-time mobile detection
